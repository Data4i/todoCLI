# todoCLI

****
## A todo CLI application
****

_**-add string**_

* Add a new TODO title

_**-del int**_

* Specify TODO to delete by index (default -1)

_**-edit string**_

* Edit a TODO title by index. id:new_title

_**-list**_

* List all TODOS

_**-toggle int**_

* Specify TODO to be toggle Completed by index (default -1)
